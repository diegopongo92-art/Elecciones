📊 Metodología de Proyección Diego Pongo Analítica

Este dashboard utiliza un modelo dinámico propio desarrollado por Diego Pongo, el cual procesa los datos públicos de la ONPE en tiempo real para proyectar resultados al 100% de actas escrutadas. No representa un resultado oficial final.

MODELO A

Proyección Lineal Directa

Extrapolación que asume que la proporción de votos restantes será idéntica a la actual. Utilizado como línea base.

⚡ MODELO B

Momentum (Tendencia Reciente)

Mide la velocidad de llegada de votos entre actualizaciones para ajustar la curva final. Corrige sesgos geográficos o de voto rural/extranjero.

Dashboard desarrollado independientemente usando datos públicos de acceso abierto. 2026.
